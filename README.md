📊 Customer Segmentation Using Machine Learning

This project implements Customer Segmentation using unsupervised machine learning (K-Means Clustering) to group customers based on purchasing behavior.

The goal is to help businesses identify distinct customer groups for targeted marketing and strategic decision-making.

📌 Project Objective

Customer segmentation helps businesses:

Identify high-value customers

Understand spending behavior

Improve targeted marketing campaigns

Increase customer retention

Enhance business profitability

This project applies clustering techniques to analyze customer data and divide customers into meaningful segments.

🧠 Technologies Used

Python

NumPy

Pandas

Matplotlib

Seaborn

Scikit-learn

📂 Dataset

The dataset includes customer information such as:

Customer ID

Gender

Age

Annual Income

Spending Score

These features are used to cluster customers based on similarities in income and spending behavior.

⚙️ Methodology
1️⃣ Data Preprocessing

Data cleaning and inspection

Feature selection

Scaling using StandardScaler (if applied)

2️⃣ Finding Optimal Clusters

Elbow Method used to determine optimal number of clusters

3️⃣ Model Implementation

K-Means Clustering algorithm from Scikit-learn

Customers grouped into distinct segments

4️⃣ Visualization

Scatter plots used to visualize customer clusters

Color-coded segments for clear interpretation

📈 Results

The model successfully segmented customers into different groups such as:

High Income – High Spending

High Income – Low Spending

Low Income – High Spending

Low Income – Low Spending

Average Income – Moderate Spending

These segments can help businesses design targeted strategies.
